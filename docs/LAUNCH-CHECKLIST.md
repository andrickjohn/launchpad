## Pre-Launch (Before First Use)

### Environment Setup
- [ ] Supabase project created (free tier)
- [ ] Supabase API keys in .env.local
- [ ] Resend account created, API key in .env.local
- [ ] Anthropic API key in .env.local (Claude Haiku)
- [ ] Reddit API credentials in .env.local
- [ ] Apify account created (for future scrapes)
- [ ] Vercel project linked and deployed

### Verification
- [ ] Auth flow works (sign up, log in, log out)
- [ ] All three screens load without errors
- [ ] CSV import works with sample data
- [ ] AI scoring returns results (test with 1 prospect)
- [ ] Email send works (test with own email)
- [ ] Campaign Wizard generates a brief
- [ ] Dashboard shows correct metrics

### Cost Milestone Map
- Day 1-7: $0-$15 (domain optional)
- First real use: ~$1-5 (Apify scrape + a few AI calls)
- Normal month: Under $10
- If usage grows significantly: evaluate Supabase Pro ($25/mo) at 500MB storage

### Acquisition Channel Validation (Day 1-2)
- [ ] Identify the primary channel for your first campaign
- [ ] Find at least 20 prospects showing intent before outreach
- [ ] Confirm direct contact path to those prospects
- [ ] Tag each prospect with their current alternative
- [ ] Test with 2-3 outreaches before scaling
