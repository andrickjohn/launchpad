# Daily Autonomous Operations — LaunchPad

## Morning Cycle (runs automatically)
1. **Check Reddit keywords** — fetch new posts matching monitored keywords
2. **Process engagement** — update open/click/reply tracking for all sent emails
3. **Fire scheduled emails** — send any sequence steps due in the next hour
4. **Recalculate metrics** — update dashboard numbers
5. **Generate daily report** — summarize yesterday's activity
6. **Check costs** — update API cost tracking, alert if approaching budget

## User Check-in (~10 minutes)
1. Review daily report
2. Review approval queue (drafts, recommendations, scrape requests)
3. Approve or adjust pending items
4. Check Overachiever suggestions

## Evening Cycle (runs automatically)
1. Re-check engagement
2. Queue follow-up reminders for tomorrow
3. Update prospect scores if new data available
4. Log daily summary to activity feed
