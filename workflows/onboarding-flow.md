# Workflow: User Onboarding

## Trigger: First login after signup

## Steps:

### Step 1: Welcome Screen
- Show welcome modal with 3-step quick start
- Option to skip (experienced users)

### Step 2: Create First Campaign
- Guide to Campaign Setup Wizard
- Prompt for: product description, target buyer, price, geography
- Wizard generates launch brief
- User reviews and confirms

### Step 3: First Prospects
- Offer three paths:
  a) Import CSV (most common)
  b) Manual entry
  c) AI "Find prospects" (generates Apify configs)
- Guide through whichever they choose

### Step 4: First Outreach
- AI generates draft emails for top-scored prospects
- User reviews drafts in queue
- Approve and send first email
- Celebrate: "Your first outreach is live!"

### Step 5: Dashboard Tour
- Quick highlight of key dashboard areas
- Show where metrics will appear after activity
- Point to Reddit monitor setup
- Mark onboarding complete
