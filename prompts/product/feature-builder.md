# Prompt: Feature Builder
# Model: Claude Sonnet (via Claude Code)
# Purpose: Guide Claude Code in building features that match the spec and pass tests.

## When Building a Feature
1. Read the relevant test spec from /qa/test-specs/
2. Read LESSONS.md for relevant past mistakes
3. Plan the implementation (components, API routes, database changes)
4. Build component by component
5. Test each component before moving on
6. Run the full feature test suite
7. Update PROGRESS.md
8. Log any lessons to LESSONS.md
